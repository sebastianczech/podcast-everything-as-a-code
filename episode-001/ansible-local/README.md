## Prerequisites

* Installed Python and [Poetry](https://python-poetry.org/docs/)
  
## Usage

1. Initialize project: `poetry init`
2. Add Ansible: `poetry add ansible`
3. Start shell: `poetry shell`
4. Execute Ansible: `ansible-playbook playbook.yml`
